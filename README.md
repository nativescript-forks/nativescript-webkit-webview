## Installation


```javascript
tns plugin add nativescript-webkit-webview
```

## Usage 


IMPORTANT: Make sure you include xmlns:wk="nativescript-webkit-webview" on the Page tag

```xml
<ui:TNSWKWebView src="https://www.google.com"/>
```
